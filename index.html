05 TERRITORY / SPLIT THE QUEST ТЕРРИТОРИЯ / ДРОБИ КВЕСТ

Don't ask for everything at once Не проси всё сразу

Don't tell AI "make me the whole zombie game". That's like asking a friend to build a whole Lego castle in one evening. Better — by parts.

Не говори ИИ "сделай мне всю игру про зомби". Это как просить друга построить целый замок из Лего за один вечер. Лучше — по частям.

  1. First: "Help me come up with the plot of a zombie game"
  2. Then: "Describe the main character"
  3. Then: "What will the levels be like?"
  4. Then: "What should the controls feel like?"
  5. Only then: "Write Scratch code for hero movement"
  1. Сначала: "Помоги придумать сюжет игры про зомби"
  2. Потом: "Опиши главного героя"
  3. Потом: "Какие будут уровни?"
  4. Потом: "Как должно выглядеть управление?"
  5. И только потом: "Напиши код для движения героя в Scratch"

At each step AI focuses on one task and crushes it. This is way better than one giant request.

На каждом шаге ИИ концентрируется на одной задаче и делает её на максимум. Это намного лучше, чем один огромный запрос.
